# Plataforma de Gestión y Monitoreo Ambiental OpenAQ

Proyecto de interfaz web estática para un dashboard de gestión y monitoreo ambiental.

## Descripción

Este proyecto consiste en una página HTML y CSS que simula un panel de control para la gestión de dispositivos IoT y monitoreo de calidad del aire con referencias a datos de OpenAQ.

El diseño incluye:
- Sidebar de navegación accesible
- Tarjetas de resumen de métricas
- Panel de mapa integrado con Google Maps
- Secciones de detalles de ubicación, gráficos y tablas
- Enfoque en accesibilidad y diseño responsivo sin JavaScript

## Estructura del proyecto

- `index (1).html`: página principal del dashboard, en español.
- `styles.css`: estilos CSS personalizados para el diseño, incluyendo paleta de colores, layout y componentes.

## Tecnologías

- HTML5
- CSS3
- Diseño responsivo
- Sin dependencias externas ni JavaScript

## Uso

1. Abrir `index (1).html` en un navegador web.
2. El dashboard se carga directamente como una página estática.

## Notas

- El mapa se integra mediante un `iframe` de Google Maps.
- Los datos mostrados son de ejemplo y simulan un entorno de monitoreo IoT.
